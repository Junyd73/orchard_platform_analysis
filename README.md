# orchard_platform_analysis

This repository is an automatically synced public analysis mirror
of selected UI source files from the private orchard_platform project.

Included:
- ui/styles.py
- ui/pages/
- ui/widgets/
- ui/components/

Excluded:
- core/
- app/
- main.py
- database files
- login/auth related files
- account/permission/admin related files
- selected internal pages
- cache files
- private docs
- secrets and environment files
